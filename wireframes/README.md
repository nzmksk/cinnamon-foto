# CinnamonFoto HTML Wireframes

This folder contains low-fidelity HTML wireframes for client review before visual design and development.

## Pages
- `index.html` — wireframe pack overview and navigation
- `home.html`
- `portfolio.html`
- `about.html`
- `services.html`
- `testimonials.html`
- `faq-terms.html`
- `contact.html`
- `booking.html`

## How to open
Option 1: Open any file directly in a browser.

Option 2: Serve locally:
```bash
python3 -m http.server 4173
```
then open `http://localhost:4173/wireframes/index.html`.

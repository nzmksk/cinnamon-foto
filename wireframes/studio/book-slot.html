Wireframe annotations
  1. Deco presets are independent — each deco has its own slot availability. Multiple clients can be in the studio simultaneously if they're in different deco setups.
  2. Client selects a deco first. The date calendar then shows availability for that deco only. Switching deco resets the date and time selection.
  3. Time slots shown are computed from: package duration + any selected add-on time + configured buffer gap. End time is displayed on each slot card.
  4. A 10-min hold timer starts on slot selection (not deco selection) to prevent double-booking while the client completes steps 2–3.
  5. Deco presets (name, description, capacity, photo) are managed by admin in Settings → Deco Presets.

Studio booking system · Raya time-slot reservation · studio subdomain
About this wireframe pack
  1. This is the studio.cinnamonfoto.co booking system — separate from the main www site.
  2. Purpose: Raya photoshoot studio time-slot reservation with online deposit payment.
  3. Slot duration is package-dependent. A configurable buffer gap separates consecutive bookings.
  4. Guest booking (no account required). Telegram notification sent on payment confirmation if client consents.
  5. Admin section manages bookings, slot availability, and system settings.

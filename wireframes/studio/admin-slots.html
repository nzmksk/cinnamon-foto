Wireframe annotations
  1. Each deco tab is an independent schedule — switching tabs shows the week view for that deco only. Bookings in different decos on the same time/date are both valid and shown separately.
  2. Admin can block a specific time range within a single deco (e.g., block Kampung Heritage 14:00–17:00 on Saturday without affecting other decos).
  3. The right panel shows a daily summary and block tool for the currently selected deco on the selected date.
  4. Buffer gaps are shown between consecutive bookings within the same deco. They do not appear between decos (they're independent).
  5. Deco tabs are generated from the list of active deco presets in Settings → Deco Presets.

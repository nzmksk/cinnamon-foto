Wireframe annotations
  1. All settings are saved globally and take effect immediately on the live booking system.
  2. Buffer gap change is the most critical setting — it recalculates available slot start times for all future bookings.
  3. Deposit % change applies to new bookings only — does not retroactively affect confirmed bookings.
  4. Package settings link to the package definitions (duration, price, inclusions) used by the slot picker.
  5. Telegram bot token is a sensitive field — masked by default. Required for reminder notifications.
